## v0.1.0 — 2025-06

### Added
- SCP policies for root account lockdown,
  region restriction, and service control
- RCP policies for resource-level controls
- Backup enforcement policies for AWS Backup
- Tag policies for mandatory resource tagging
- Declarative policies for EC2 governance
- AI service opt-out policies for AWS AI/ML
- Apache 2.0 license with Vijenex™ trademark notice
- CONTRIBUTING.md with submission guidelines
- Validation script for local policy checking
- GitHub Actions workflow for PR validation
